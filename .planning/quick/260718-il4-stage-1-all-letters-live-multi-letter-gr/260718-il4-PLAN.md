   `FutureProvider` hardwired to the single `assets/curriculum/curriculum_graph.json`

   Python script, so a thaa unit runs end-to-end exactly like baa: reachable from the
   Journey map, presenting the full Phase-19 question presentation (instruction bar +
   stimulus zone) through the live `presentGraphExercise` seam.

for the remaining 24 letters and only then rebuilds per-letter server data.

per-letter graph provider, thaa reachability, an extended learned-letters lint, and a
live-path widget test proving a thaa unit mounts and presents.

- Micro-drills stay OUT of every graph (thaa draft carries none — keep it that way).

  `server/app/curriculum_data/generate.py` — per-letter server data is Stage 2.
