### Two grounding-seam pitfalls the matrix bakes in (verified CONFIRMED)
- **structured-output + function-calling throws** (`application/json` mime exception) → FACTS must be injected as system-instruction/Content **text**, never as a responseSchema, alongside the 4 tools. This actually *reinforces* FACTS-in-as-text / ACTIONS-out-as-tools.
- **streaming + tool-calls** is **undocumented on Dart** (⚠︎ verifier softened "unsupported"→"don't rely on streamed function-call deltas") → run the ACTION loop on non-streamed `chat.sendMessage` turns; stream only the final `say()` text turn, or use Live API.
