# Phase 02.1.1 Plan 03: Demo Home + Watch Screens Summary

**The first two walkthrough screens: a de-gamified demo Home (idle mascot + warm greeting + alif lesson card → Watch) and a Watch stroke-order screen (write mascot + dotted alif painted from the single reference source + numbered gold start-dot + Start Tracing → Trace), both wired into their fixed /demo route slots.**

- **Duration:** ~25 min
- **Started:** 2026-06-01
- **Tasks:** 2/2 (both TDD: RED → GREEN)

## What shipped

### Task 1 — Demo Home (de-gamified, DP-04/DP-06)
- `DemoHomeScreen`: a parchment Scaffold with the idle-pose `QalamMascot` greeting the child (`demoHomeGreeting` = "Let's learn, Layla."), and a single raised alif **"Today's Lesson"** card (`demoLessonEyebrow` + the alif glyph via `ArabicText` display island + `demoLessonTitle`) that taps through to `/demo/watch`. >= targetLarge touch target, sticker-shadow surface.
- **Anti-gamification proven (BINDING):** Test 3 asserts NO `Text` matching `stars/weekly/streak/badge/+N` and the source carries no `star.svg`. The mockup's header star counter, weekly tally, and 3-star rating are all omitted.
- Wired `/demo/home` → `DemoHomeScreen` via a `_screenFor(DemoStep)` dispatch (remaining steps still fall back to the calm placeholder).

### Task 2 — Watch (stroke order, DP-03/DP-07)
- `DemoWatchScreen`: eyebrow/heading (`demoWatchEyebrow` / `demoWatchHeading`), the write-pose mascot beside a soft-aqua-framed parchment canvas card, a calm tip card (`demoWatchTip`), and a keyed "Start Tracing" sticker CTA → `/demo/trace`.
- `DottedAlifPainter` (public): paints the alif as a stippled dotted line from `DemoAlif.referencePoints` (one source of truth — never `Text('ا')`) with a numbered **gold** start-dot at the top via `QalamColors.reward` (the one sanctioned gold use here). The guide sits under an active `IgnorePointer` — watch, don't touch.
- **No audio chrome (DP-07):** Test 4 asserts no `Listen` / `Play sound` text.
