## What Was Built

### Task 1: JourneyProgress Model + JourneyNodeState Enum (TDD)

**RED phase:** Wrote `test/models/journey_progress_test.dart` covering all four `JourneyNodeState.compute()` outcomes (complete, current, future, locked edge guard) and `JourneyProgress` construction. Tests confirmed failing before implementation.

**GREEN phase:** Created `lib/models/journey_progress.dart`:
- `enum JourneyNodeState` with four values: `complete`, `current`, `future`, `locked`
- `static JourneyNodeState compute(String letterId, Set<String> masteredIds, String currentId)` — returns `locked` for empty string, `complete` for mastered, `current` for current letter, `future` otherwise
- `class JourneyProgress` — immutable, `const` constructor, `masteredIds` + `currentId` fields
- `JourneyProgress.empty()` factory returning the unauthenticated/initial state

All 6 tests passed green.
