# Phase 03.1 Plan 02: Journey Map Screen ‚Äî Visual Implementation Summary

Full JourneyScreen with winding path CustomPainter, 28 positioned letter nodes at pixel-exact coordinates, four visual states, TODAY pill, and Level 1 header pill ‚Äî replacing the plan 01 placeholder.

## What Was Built

### Task 1: JourneyPathPainter

Created `lib/features/journey/widgets/journey_path_painter.dart`:

- `JourneyPathPainter extends CustomPainter` with `const` constructor taking `completedCount`.
- **Grey full trail:** `_buildFullPath()` produces a `Path` matching the SVG in `journey_preview.html` exactly ‚Äî moveTo (1094,160), lineTo, quadraticBezierTo corners at row transitions, cubicTo checkpoint (590,660). Paint: `QalamColors.border` (#D6E8E8), strokeWidth 22, round caps/joins.
- **Green completed overlay:** Horizontal line from (1094,160) to the x-position of the last complete letter in Row 1. For mock state (completedCount=3): from (1094,160) to (758,160). Paint: `QalamColors.success` at 60% opacity, strokeWidth 22, round caps.
- `shouldRepaint` checks `completedCount` change only.

### Task 2: JourneyScreen ‚Äî Full Implementation

Replaced the plan 01 placeholder in `lib/features/journey/journey_screen.dart`:

**Data layer:**
- `ConsumerWidget` reads `mockJourneyProgressProvider` via `ref.watch`.
- `_kLetters` ‚Äî static const list of 28 Arabic letters in curriculum order (Alif ‚Üí Ya) with id, glyph, name.

**Layout:**
- `Scaffold(backgroundColor: QalamColors.bg)` with `SafeArea` and a full-screen `Stack`.
- `_nodePosition(index)` computes pixel coordinates: rows 0/2 are R‚ÜíL (`_kXPositions[6 - col]`), rows 1/3 are L‚ÜíR (`_kXPositions[col]`).
- 28 `Positioned` nodes at `(pos.dx - 34, pos.dy - 34)` ‚Äî center-aligned for 68px circles.

**Node visual states (4):**
- `complete` ‚Äî `QalamColors.success` fill, `Color(0xFF2A8A60)` drop-shadow at (0,5), white glyph.
- `current` ‚Äî `QalamColors.primary` fill, `QalamColors.primaryPressed` drop-shadow at (0,5), white glyph + TODAY pill.
- `future` ‚Äî `_DashedCirclePainter` overlay on white 68√ó68 SizedBox, muted glyph (`QalamColors.fgMuted`).
- `locked` ‚Äî `Color(0xFFCDD8DA)` warm grey fill, muted glyph (edge case defensive state).

**Private widgets:**
- `_JourneyNode` ‚Äî Column(circle + name label). Uses a SizedBox height wrapper (96px for current, 75px for others) with `Stack(clipBehavior: Clip.none)` so the TODAY chip floats above the circle.
- `_NodeCircle` ‚Äî 68√ó68 Container with state-dependent decoration; future nodes use `CustomPaint(_DashedCirclePainter)` instead.
- `_DashedCirclePainter` ‚Äî `canvas.drawArc` loop; dashLength=6, gapLength=4, strokeWidth=2.5, color=`QalamColors.border`.
- `_TodayChip` ‚Äî Teal pill "TODAY" in 10pt Nunito 700, letterSpacing 1.0.

**Level 1 header pill:**
- `Positioned(top: 20, left: 0, right: 0)` ‚Üí `Center` ‚Üí Container with teal-tint background + ink-teal border + pill radius.
- Row: "Level 1" (Fredoka 700 20px) | "LETTERS" (label+letterSpacing) | 1px divider | "N of 28 mastered" (computed from `progress.masteredIds.length`).

**Anti-gamification (D-23/D-24):** No star counter, no streak, no "+N" copy, no `QalamColors.reward` (gold) anywhere on screen.
