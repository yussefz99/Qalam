# Phase 03.1 Plan 03: Journey Map Screen — Animation, Tap Handlers, Star Badges Summary

JourneyNodeWidget with 2400ms pulse glow animation, gold 3-star badge on mastered nodes, tap routing to /practice for unlocked letters, Level 1 Quiz checkpoint, and Level 2 locked banner — completing the Journey Map Screen feature.

## What Was Built

### Task 1: JourneyNodeWidget (animation + star badge + tap handlers)

Created `lib/features/journey/widgets/journey_node_widget.dart`:

- `JourneyNodeWidget extends StatefulWidget` — replaces the plan 02 `_JourneyNode` StatelessWidget.
- **Pulse animation:** `AnimationController(duration: 2400ms)` + `CurvedAnimation(Curves.easeInOut)` + `repeat(reverse: true)`. Only active when `state == NodeState.current`. `AnimatedBuilder` drives a continuously varying `BoxDecoration` spread/blur on the aqua ring around the current node.
- **Gold star badge (`_StarBadge`):** Three `_StarPainter` CustomPainter instances arranged in a cluster, positioned bottom-right of complete nodes. Uses `QalamColors.reward` (gold) — the sole permitted reward-color use on this screen per D-23/D-24.
- **Tap handlers:** `GestureDetector.onTap = null` for `future` and `locked` nodes (inert, no navigation, no feedback). `context.go('/practice')` for `complete` and `current` node taps.
- **Co-located widgets:** `_DashedCirclePainter` and `_TodayChip` moved from `journey_screen.dart` into this file for logical co-location.
- **Memory safety:** `dispose()` calls `_animController.dispose()` to prevent leaks when the widget leaves the tree.

### Task 2: Journey Screen — Checkpoint and Level 2 Banner

Updated `lib/features/journey/journey_screen.dart`:

- **Level 1 Quiz checkpoint box:** `Positioned` at y≈660 — a rounded container with lock icon and "Level 1 Quiz" label. Milestone marker, not a gamification element.
- **Level 2 locked banner:** `Positioned` at y≈752 with `Opacity(0.72)` — solid border, "Level 2 — Coming Soon" treatment. Signals future progression without acting as a reward lure.
- **Cleanup:** Removed `_JourneyNode`, `_NodeCircle`, `_DashedCirclePainter`, `_TodayChip` private classes (now in `journey_node_widget.dart`). Screen reduced from 399 lines to layout + path painter concerns only.

## Anti-Gamification Confirmation (D-23/D-24)

- No running star counter anywhere on screen.
- No streak display, no "+N" copy, no leaderboard hint.
- `QalamColors.reward` (gold) appears **only** on the 3-star badge on mastered nodes — information ("you truly mastered this letter"), not score.
- Checkpoint box and Level 2 banner are structural/navigational, not reward elements.
