|----------|-------------|
| client → Firebase AI Logic (network) | The ONLY network egress of the tutor. Only TutorFacts (derived non-PII) cross; the Gemini key stays server-side behind the proxy; App Check gates the caller. |
| Gemini response → dispatcher | Model output returns as one of 4 ACTION tool calls (pinned by .any({4})); it supplies a LINE, never a verdict/star. |
